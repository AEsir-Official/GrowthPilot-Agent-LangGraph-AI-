# Resume Bullets

## 中文项目名

GrowthPilot Agent：AI 增长实验设计系统

## 中文项目描述

基于 LangGraph 构建面向消费 / 电商场景的多 Agent 工作流系统，将商业想法自动拆解为业务类型识别、转化漏斗建模、A/B 测试设计、需求池生成、PRD 初稿、埋点方案、指标体系和迭代建议。

## English Project Name

GrowthPilot Agent: AI Growth Experiment Design System

## English Project Description

Built a multi-agent workflow system with LangGraph to transform business ideas into conversion funnels, A/B testing plans, requirement pools, PRD drafts, event tracking plans, metric systems, and iteration suggestions.

## 简历 Bullet

1. 基于 Python、Streamlit 和 LangGraph 搭建多 Agent 工作流，将消费 / 电商商业想法拆解为业务类型、用户画像、转化漏斗、增长实验、需求池、PRD、埋点和指标体系。
2. 设计 OpenAI-compatible API 调用层和 mock fallback 机制，在无 API key 或 API 调用失败时仍可稳定生成完整演示结果，提升项目可展示性。
3. 使用本地 markdown 知识库和 TF-IDF RAG 检索业务模板，结合 Critic Agent 输出 badcase 分析和下一轮迭代建议，减少空泛生成。
