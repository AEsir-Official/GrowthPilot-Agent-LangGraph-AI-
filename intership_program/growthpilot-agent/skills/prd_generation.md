# skill name

prd_generation

## purpose

将需求池和增长实验整理为结构化 PRD 初稿。

## input

- requirement pool
- funnel summary
- experiment design

## output

- project background
- target users
- core path
- functional requirements
- non-functional requirements
- acceptance criteria
- risk boundaries

## related agent

PRD Agent

## evaluation criteria

- 是否包含完整 PRD 章节
- 是否能从业务目标追溯到功能需求
- 是否有验收标准和数据指标
- 是否定义版本边界

## example

输入：大学生在线考证课程平台

输出：包含背景、用户痛点、试听预约路径、功能需求、指标和风险边界的 PRD 初稿。
