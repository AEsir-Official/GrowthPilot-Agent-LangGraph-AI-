# skill name

metrics_design

## purpose

设计北极星指标、一级指标、过程指标和实验判断指标。

## input

- business type
- funnel output
- event tracking plan

## output

- north star metric
- key funnel metrics
- experiment evaluation metrics
- quality metrics

## related agent

MVP Agent

## evaluation criteria

- 是否有明确北极星指标
- 是否与漏斗环节匹配
- 是否支持 A/B 测试判断
- 是否避免指标过多导致重点不清

## example

输入：平价护肤电商平台

输出：北极星指标为新用户首单转化数，一级指标包括 CTA 点击率、加购率、首单转化率。
