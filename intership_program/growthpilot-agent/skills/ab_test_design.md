# skill name

ab_test_design

## purpose

围绕漏斗瓶颈生成可执行的增长实验和 A/B 测试方案。

## input

- funnel analysis
- business goal
- retrieved experiment template

## output

- experiment hypothesis
- control vs treatment design
- core metrics
- observation metrics
- success criteria
- risk notes

## related agent

Experiment Agent

## evaluation criteria

- 是否包含实验组和对照组
- 是否有核心指标和观察指标
- 是否有成功标准和风险
- 是否与具体漏斗环节绑定

## example

输入：平价护肤电商首页转化低

输出：首屏价值主张 A/B 测试，比较“立即购买”与“测测我的肤质方案”两个 CTA，并用 CTA 点击率和首单转化率验证。
