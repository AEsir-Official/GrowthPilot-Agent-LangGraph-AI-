# skill name

event_tracking

## purpose

为增长实验和关键漏斗环节设计埋点事件。

## input

- funnel stages
- experiment goals
- mvp feature list

## output

- event_name
- trigger
- properties
- purpose
- related_metric

## related agent

MVP Agent

## evaluation criteria

- 是否覆盖关键漏斗环节
- 是否包含必要属性
- 是否能服务具体指标分析
- 是否支持实验版本对比

## example

输入：校园二手交易平台

输出：`publish_cta_click`、`publish_submit`、`chat_click`、`trade_confirm` 等事件及对应属性。
