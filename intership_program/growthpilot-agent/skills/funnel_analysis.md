# skill name

funnel_analysis

## purpose

将商业想法拆解为转化漏斗，明确每个环节的用户行为、业务问题、核心指标和可优化方向。

## input

- business idea
- target user context
- retrieved knowledge base context

## output

- conversion funnel
- stage-level metrics
- optimization opportunities

## related agent

Funnel Agent

## evaluation criteria

- 是否覆盖完整漏斗阶段
- 是否明确用户行为和业务问题
- 是否为每个阶段绑定核心指标
- 是否提出可验证优化方向

## example

输入：校园二手交易平台

输出：访问 -> 浏览商品 -> 发布 / 咨询 -> 成交，并为每个环节给出发布完成率、咨询率、成交率等指标。
